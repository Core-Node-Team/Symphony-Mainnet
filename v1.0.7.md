## BEFORE UPGRADE BLOCK
```
cd $HOME
rm -rf symphony
git clone https://github.com/Orchestra-Labs/symphony
cd symphony
git checkout v1.0.7
make build
```
```
mkdir -p $HOME/.symphonyd/cosmovisor/upgrades/v1.0.7/bin
mv $HOME/symphony/build/symphonyd $HOME/.symphonyd/cosmovisor/upgrades/v1.0.7/bin/
```
```
$HOME/.symphonyd/cosmovisor/upgrades/v1.0.7/bin/symphonyd version
```
